# n8n Workflows

This directory stores exported n8n workflow JSON files.

## How to Export from n8n
1. Open n8n
2. Open the workflow
3. Click ··· → Export → Download JSON
4. Save the file here as `<workflow-name>.json`

## How to Import
1. Open n8n
2. Click + New Workflow
3. ··· → Import from File → select the JSON
