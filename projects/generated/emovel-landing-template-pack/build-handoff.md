# Build Handoff: EMOVEL Landing Template Pack

## Product Summary

EMOVEL Landing Template Pack is a $97 template product that gives buyers premium Next.js landing page templates built with Tailwind, TypeScript, reusable sections, and deployment-ready structure.

## Route Structure

- `/` - product sales page.
- `/checkout` - manual Gumroad handoff.
- `/thank-you` - delivery instructions.
- `/privacy` - optional policy.

## Component Hierarchy

- AppShell
- HeroSection
- TemplatePreviewGrid
- IncludedSectionsList
- TechStackSection
- DeploymentSteps
- PricingSection
- FAQSection
- FinalCTA

## Design Direction

Use a premium UI kit feel: crisp previews, file panels, graphite accents, blue CTA, mint success states, and clear deployment flow.

## Acceptance Checklist

- [ ] Template variants are defined.
- [ ] Sales page explains code deliverables.
- [ ] Price is visible.
- [ ] Gumroad handoff is manual.
- [ ] Page builds locally.
