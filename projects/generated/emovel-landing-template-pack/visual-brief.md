# visual-brief

## Visual Direction

The Landing Template Pack should feel like a premium UI kit for operators: clean, structured, fast, and deployable.

## Palette

- Primary Ink: `#101114`
- Deep Graphite: `#20242B`
- Signal Blue: `#2F6BFF`
- Electric Mint: `#40D9A3`
- Soft Cloud: `#F5F7FA`
- White: `#FFFFFF`
- Line Gray: `#D9DEE7`

## Typography

- Headings: Inter Tight or Satoshi.
- Body: Inter.
- Labels and file paths: IBM Plex Mono.

## Layout

- Use full-width section bands.
- Use preview cards for repeated templates only.
- Keep radius 8px or less.
- Make code/file structure visible.
- Use crisp screenshots or generated preview imagery when available.

## Imagery

Use template previews, code file panels, section thumbnails, and deployment flow diagrams. Avoid abstract imagery and generic SaaS gradients.
