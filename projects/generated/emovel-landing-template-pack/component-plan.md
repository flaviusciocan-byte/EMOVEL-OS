# component-plan

## Route Structure

- `/` - sales page for the template pack.
- `/checkout` - manual Gumroad handoff.
- `/thank-you` - download and setup instructions.
- `/privacy` - optional policy route.

## Component Hierarchy

- AppShell
- HeroSection

- IncludedSectionsList

## Template Components

The product itself should include reusable sections:

- Hero
- LogoProofStrip
- BenefitsGrid
- FeatureRows
- PricingCards
- FAQ
- CTASection
- Footer

## Data Model

Keep all template metadata local. No CMS, API, database, or payment integration required for v1.
