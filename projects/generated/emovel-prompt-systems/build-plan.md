# build-plan

## App Target

Create or update a Next.js, TypeScript, Tailwind product page for EMOVEL Prompt Systems.

## Required Files

- `package.json`
- `app/layout.tsx`
- `app/page.tsx`
- `app/checkout/page.tsx`
- `app/thank-you/page.tsx`
- `app/privacy/page.tsx`
- `app/globals.css`
- `components/`
- `lib/content.ts`

## Build Steps

1. Move product copy into local content constants.
2. Build a static landing page using the component plan.
3. Add prompt system anatomy as a compact table or structured panel.
4. Add included systems grid.
5. Add pricing and FAQ.
6. Run `npm install`.
7. Run `npm run build`.

## Constraints

- No paid APIs.
- No database.
- No auth.
- No Gumroad API.
- Local static content only.
