# visual-brief

## Visual Direction

Prompt Systems should look like a clean AI operations manual: structured, sharp, calm, and useful.

## Palette

- Primary Ink: `#101114`
- Deep Graphite: `#20242B`
- Signal Blue: `#2F6BFF`
- Electric Mint: `#40D9A3`
- Soft Cloud: `#F5F7FA`
- White: `#FFFFFF`
- Line Gray: `#D9DEE7`

## Typography

- Headings: Inter Tight or Satoshi.
- Body: Inter.
- Labels/code: IBM Plex Mono.

## Layout

- Strong text hierarchy.
- Compact tables for prompt anatomy.
- Cards only for repeated systems.
- Use code-style panels for examples.
- Keep radius at 8px or less.

## Imagery

Use concrete visuals:

- prompt documents
- workflow boards
- AI output checklists
- operator desk with structured notes
- system diagrams

Avoid robot faces, neon AI art, abstract blobs, and purple gradients.
