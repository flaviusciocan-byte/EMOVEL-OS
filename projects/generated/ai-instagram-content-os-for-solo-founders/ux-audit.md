# UX audit

**Experience goal:** Make the interface feel like a premium production cockpit, not a generic prompt box.

**Checks:**
- One primary action per view.
- Large prompt input remains the visual anchor.
- Output panels are scannable, exportable, and clearly sourced.
- Touch targets are at least 44px high.
- Empty state explains what to do next without tutorial clutter.

**Success metric:** purchase intent.
