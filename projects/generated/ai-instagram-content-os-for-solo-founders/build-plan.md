# build plan

**App target:** Local Next.js + Tailwind + TypeScript app.

**Implementation steps:**
1. Create typed local templates.
2. Build prompt input and selectors.
3. Generate markdown client-side.
4. Render output panels.
5. Add export markdown button.
6. Run install, build, and smoke test.

**No paid API calls in v1.**
