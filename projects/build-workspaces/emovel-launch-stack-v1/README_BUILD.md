# README_BUILD: EMOVEL Launch Stack v1

This builder workspace contains the local handoff files for EMOVEL Launch Stack v1.

## Use With GPT-Pilot / Pythagora

1. Open GPT-Pilot or Pythagora manually.
2. Paste `gpt-pilot-prompt.md` as the project brief.
3. Target `products/emovel-launch-stack-v1/landing-app/` only if intentionally updating the product app.
4. Run build checks manually.

Prompt Studio does not run GPT-Pilot, execute shell commands, call paid APIs, or connect Gumroad.

## Expected Output

A production-ready landing app and Gumroad-ready publish package for the first EMOVEL shop product.
