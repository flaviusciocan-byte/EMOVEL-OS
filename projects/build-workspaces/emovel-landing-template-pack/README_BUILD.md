# README_BUILD: EMOVEL Landing Template Pack

This builder workspace contains the local handoff files for EMOVEL Landing Template Pack.

## Use With GPT-Pilot / Pythagora

1. Open GPT-Pilot or Pythagora manually.
2. Paste `gpt-pilot-prompt.md` as the project brief.
3. Target `apps/emovel-site/app/products/landing-template-pack/` only if intentionally updating the public product page.
4. Package template files separately before publishing.
5. Run build checks manually.

Prompt Studio does not run GPT-Pilot, execute shell commands, call paid APIs, or connect Gumroad.
