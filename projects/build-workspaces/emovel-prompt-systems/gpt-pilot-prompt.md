# GPT-Pilot / Pythagora Build Prompt: EMOVEL Prompt Systems

Build a polished Next.js, TypeScript, Tailwind product page for EMOVEL Prompt Systems.

## Product Goal

Sell a $197 product that gives buyers production-grade prompt frameworks for copywriting, research, positioning, strategy, and content production.

## Target User

Founders, consultants, creators, and operators who use AI often but struggle with inconsistent results, repeated prompt rewriting, and lack of documented workflows.

## Route Structure

- `/` primary sales page
- `/checkout` manual Gumroad handoff placeholder
- `/thank-you` delivery instructions
- `/privacy` optional policy route

## Component Hierarchy

- AppShell
- HeroSection
- ProblemSection
- SystemAnatomySection
- IncludedSystemsGrid
- UseCasesSection
- PricingSection
- FAQSection
- FinalCTA

## Styling Rules

Use the EMOVEL palette and make the page feel like a clean AI operations manual. Use code-style panels, compact tables, and clear CTAs. Avoid robot imagery, neon AI visuals, and purple gradients.

## Build Constraints

- No paid APIs.
- No database.
- No Gumroad API.
- Static local content only.
- Must pass `npm run build`.
