# README_BUILD: EMOVEL Prompt Systems

This builder workspace contains the local handoff files for EMOVEL Prompt Systems.

## Use With GPT-Pilot / Pythagora

1. Open GPT-Pilot or Pythagora manually.
2. Paste `gpt-pilot-prompt.md` as the project brief.
3. Target `apps/emovel-site/app/products/prompt-systems/` only if intentionally updating the public product page.
4. Run build checks manually.

Prompt Studio does not run GPT-Pilot, execute shell commands, call paid APIs, or connect Gumroad.
