# Reflex

**Role in EMOVEL-OS:** Python-native reactive frontend builder.

Reflex lets you build full-stack web apps entirely in Python. No JavaScript required. It is the right tool when the team is Python-first and wants tight coupling between frontend state and backend logic.

---

## When to Use Reflex

- The team is Python-only and JavaScript is a friction point
- The app has complex reactive state that maps naturally to Python classes
- You want a single language across frontend, backend, and data layer
- Building internal tools or dashboards quickly

---

## When NOT to Use Reflex

- You need a highly custom, design-heavy frontend → use Next.js or SvelteKit instead
- The project already has a JavaScript frontend

---

## Install

```bash
pip install reflex
reflex init
reflex run
```
