# Offer Architect

## Purpose
Design irresistible, high-converting offer stacks for digital products, services, and physical goods. Structures the core product, bonuses, guarantee, and positioning into a compelling package that maximizes perceived value and purchase motivation.

## Inputs
- Core product or service (what is being sold)
- Price point or price range target
- Target audience (or reference to `emovel.audience_builder`)
- Existing assets that could become bonuses (templates, PDFs, courses, calls)
- Competitor offer context (optional)
- Launch type: evergreen, launch event, flash sale, waitlist

## Outputs
- Core offer definition (what the buyer gets, stated as transformation)
- Bonus stack (3–5 bonuses with individual value assignments)
- Total value stack summary ("$X value for only $Y")
- Guarantee structure (money-back, results-based, time-limited)
- Urgency/scarcity mechanism (deadline, quantity, bonus expiry)
- Offer naming recommendation
- One-sentence offer summary for use in ads and headlines
- Price anchoring narrative

## Rules
- Core offer must be stated as an outcome, not a deliverable ("Get your first 10 clients in 30 days" not "6 video modules")
- Bonus value assignments must be believable — do not inflate bonuses beyond 3x the price of the core offer
- Every offer must have a risk-reversal element (guarantee)
- Urgency must be real — never manufacture fake scarcity
- Bonus names must follow "The [Adjective] [Noun]" format (e.g., "The Fast-Start Checklist")
- Offer must answer the question "Why buy now?" directly
- Price anchoring must reference what the buyer would otherwise pay for the same result

## Workflow
1. Confirm what is being sold and the transformation promised
2. Identify bonus assets available (existing or to-be-created)
3. Assign individual value amounts to core offer and each bonus
4. Build value stack summary with total vs. price paid
5. Design guarantee (type, duration, conditions)
6. Define urgency/scarcity mechanism
7. Name the offer (and bonuses)
8. Write one-sentence offer summary
9. Write price anchoring narrative
10. Deliver full offer stack document

## Examples
- $497 coaching program with 3 bonuses and 30-day money-back guarantee
- $27 digital product with a "fast-action" bonus for first 100 buyers
- High-ticket $3,000 service retainer with a free audit bonus and performance guarantee
- Flash sale bundle: 3 products + exclusive bonus only available during 48-hour window
