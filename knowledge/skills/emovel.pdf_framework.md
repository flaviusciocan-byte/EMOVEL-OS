# PDF Framework

## Purpose
Design and produce structured, branded PDF documents — lead magnets, guides, workbooks, checklists, SOPs, and ebooks — that deliver standalone value and support EMOVEL offer stacks.

## Inputs
- Document type: lead magnet, guide, workbook, checklist, SOP, ebook, case study
- Topic and core promise of the document
- Target audience (or reference to `emovel.audience_builder`)
- Length target (pages or word count)
- Brand kit: colors, fonts, logo (or reference to `emovel.visual_brief`)
- Distribution purpose: opt-in freebie, paid bonus, onboarding asset, sales tool

## Outputs
- Full document outline (section headings and subheadings)
- Complete written content per section
- Design layout specs (grid, color blocks, typography hierarchy)
- Cover page copy and design notes
- Call-to-action placement within document
- Export-ready Markdown or structured content for PDF tools (Canva, Adobe, Notion)

## Rules
- Every PDF must have a transformation promise on page 1
- Sections must be scannable: use headers, bullets, numbered lists
- Include at least one actionable exercise or template per major section
- Brand colors and logo must appear on cover, footer, and last page
- CTAs embedded within the document must be subtle (resource box or footer link)
- PDFs used as lead magnets must deliver the promised result within the document itself — no cliffhangers
- Workbooks require blank fill-in sections; checklists require checkbox formatting

## Workflow
1. Confirm document type and distribution purpose
2. Define core transformation promise
3. Build full outline (cover → intro → 3–7 main sections → summary → CTA page)
4. Write each section following outline
5. Specify layout notes per section (visual callouts, diagrams, tables)
6. Design cover page copy and visual direction
7. Place CTAs at natural close points (end of sections, last page)
8. Deliver structured content ready for layout in Canva/Adobe/Notion

## Examples
- "10 Client Scripts That Close" — lead magnet checklist PDF
- "The 30-Day Content Calendar Workbook" — paid bonus for a course
- "EMOVEL Onboarding SOP" — internal operations document
- "Case Study: How [Client] 10x'd Revenue in 90 Days" — sales tool PDF
