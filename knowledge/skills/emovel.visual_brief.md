# Visual Brief

## Purpose
Produce a complete visual direction brief that aligns design, brand, and creative assets across all EMOVEL productions — ensuring visual consistency between pages, PDFs, ads, social content, and product assets.

## Inputs
- Brand name and industry/niche
- Target audience (or reference to `emovel.audience_builder`)
- Desired brand feeling: words (e.g., bold, minimal, luxe, playful, clinical)
- Existing assets: logo, fonts, color hex codes (if available)
- Content type the brief is for: website, social media, PDF, ad creative, product packaging
- Competitor visual reference (optional: describe or list)
- Budget tier: DIY (Canva), mid (Figma/Webflow), premium (custom design)

## Outputs
- Brand personality statement (3–5 sentences)
- Color palette (primary, secondary, accent, neutral) with hex codes
- Typography system (heading font, body font, accent font) with usage rules
- Visual style descriptor (e.g., "dark luxury editorial," "clean SaaS minimal," "warm earthy wellness")
- Image/photography direction (subject, lighting, composition, mood)
- Iconography and illustration style
- Spacing and layout principles
- DOs and DON'Ts for visual usage
- Moodboard description (6–9 reference image directions)
- Platform-specific format specs (social sizes, PDF page dimensions, ad specs)

## Rules
- Color palette must pass WCAG AA contrast ratio for text on background
- Maximum 2 display fonts and 1 body font per brand system
- Visual style must align with audience trust signals (e.g., clinical audiences expect clean lines, not decorative scripts)
- Photography direction must specify human presence: people-forward, product-only, or lifestyle context
- Every visual brief must include a "what NOT to do" section to prevent off-brand creative
- Budget tier determines tool recommendations — never recommend premium tools for DIY budgets

## Workflow
1. Receive inputs (brand name, audience, feeling words, existing assets)
2. Define brand personality in 3–5 sentences
3. Build color palette with hex codes and usage roles
4. Select typography system with pairing rationale
5. Write visual style descriptor
6. Define photography and image direction
7. Specify iconography style
8. Write layout and spacing principles
9. List DOs and DON'Ts
10. Describe moodboard reference directions
11. Add platform format specs relevant to the content type

## Examples
- Visual brief for a $500K+ personal brand coach entering high-ticket market
- Brand system for a Shopify dropshipping store targeting Gen Z
- PDF and slide deck visual brief for a B2B SaaS company
- Social media creative brief for a wellness supplement brand
