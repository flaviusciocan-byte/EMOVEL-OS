## Purpose
Free, open-source screen recording Chrome extension with annotation tools. Records screen, tab, or camera, with drawing, blur, and highlighting overlays — no upload or account required. Everything stays local.

## Use Cases
- Quick annotated screen recordings for bug reports and feedback
- Record UI walkthroughs with pen/highlight annotations for clarity
- Capture browser-based tool demos without leaving the browser
- Create annotated tutorial clips to embed in product onboarding
- Low-friction content creation: no app to install, works instantly in Chrome

## Installation Path
```
Chrome Web Store → search "Screenity" → Add to Chrome
```
Or install from source:
```bash
git clone https://github.com/alyssaxuu/screenity.git
# Load as unpacked extension in Chrome developer mode
```

## Dependencies
- Google Chrome or Chromium-based browser (Brave, Edge)
- No server, no account, no cloud — fully local
- Storage: recordings saved to local disk or exported as WebM/MP4

## Recommended Integrations
- **Cap** — use Screenity for quick annotated clips; Cap for shareable async demos
- **Content Engine** — export clips and use as raw content for social posts
- **OpenCut** — edit Screenity recordings in OpenCut before publishing
- **Postiz** — upload edited clips to schedule across platforms

## Notes
Screenity is the zero-friction option — open a tab and record in seconds. Use it for annotations and quick captures. For shareable links and async review, upgrade to Cap. Not suited for long-form recordings or anything requiring persistent hosting.
